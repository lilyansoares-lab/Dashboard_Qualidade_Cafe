# ☕ Dashboard de Análise: Qualidade do Café no Mercado Global

Este projeto apresenta uma análise exploratória de dados combinada com técnicas de **Data Storytelling** para entender os fatores que definem a excelência dos cafés especiais ao redor do mundo, baseando-se nos critérios da SCA (Specialty Coffee Association)[cite: 1].

## 🎯 Principais Insights da Análise

* **Domínio de Mercado:** A espécie **Arábica** domina amplamente o cenário de cafés especiais, representando **97.9%** dos dados analisados[cite: 1].
* **Padrões de Qualidade:** Avaliação detalhada de como as notas de sabor e aroma se correlacionam e quais países mantêm maior consistência nas safras[cite: 1].
* **Fator Umidade:** Mapeamento da umidade ideal por continente para garantir a integridade do grão durante os processos de exportação[cite: 1].

## 🛠️ Tecnologias e Conceitos Utilizados
* **Looker Studio (Google Data Studio):** Construção de dashboards interativos e dinâmicos.
* **Data Storytelling:** Design de painéis focado em negócios, fluxo narrativo linear e eliminação de ruídos visuais.
* **Análise de Dados:** Estruturação de métricas operacionais e de qualidade de produto[cite: 1].
